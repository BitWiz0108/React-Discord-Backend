# React Discord API

> This is the Back-end of React-Discord website written in Typescript using Node.js, Socket.io and MySQL.
This website is developed from ground up using Node.Js.

## Stack
- Node
- Socket.io
- MySQl
- Eslint Airbnb
- Prettier
- Yarn
- Npm
- Husky - Git Hooks
- Fetch - HTTP Request
- Install
- yarn
- yarn start
